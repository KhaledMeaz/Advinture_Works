# Advinture_Works

This is a Data Analysis and Visualization project based on the **Advinture Works 2019 Dataset**.

## Project Overview
- Cleaned and preprocessed 7 years of retail sales data.
- Created **interactive Power BI dashboards** to analyze:
  - Sales by Region, State, and City
  - Sales by Category and Sub-Category

## Tools Used
- Power BI
- Excel/CSV
